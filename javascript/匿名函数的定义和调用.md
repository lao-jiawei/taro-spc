匿名函数

* `function`关键字后没有函数名（即标识符）的函数

1. 定义匿名函数

   ````javascript
   var sayHi=function(){
     alert('Hi!');
   }
   ````

2. 调用匿名函数

> PS：必须在使用前先赋值，否则会报错
>
> 如：
>
> ````javascript
> sayHi();	//报错：函数还不存在
> let sayHi=function(){
>   alert("Hi!");
> }
> ````

* 正确使用匿名函数

  ````javascript
  let sayHi=function(){
    alert("Hi!");
  }
  sayHi();
  ````

* 条件判断使用不同的匿名函数

  ````javascript
  let sayHi;
  if(条件){
    sayHi=function(){
      console.log("Hi!");
    }
  }else{
    sayHi=function(){
      console.log("Yo!");
    }
  }
  ````
