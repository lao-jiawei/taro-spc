1. 函数声明

   ````javascript
   function functionName(arg0,arg1,arg2){
     //函数体
   }
   ````

   `function`：关键字

   `{functionName}`：函数名

   > 特征：函数声明提升
   >
   > * *即在执行代码之前会先读取函数声明，可以把函数声明放在调用它的语句后面*
   >
   >   ````javascript
   >   sayHi();
   >   function sayHi(){
   >     alert('Hi!')
   >   }
   >   ````
