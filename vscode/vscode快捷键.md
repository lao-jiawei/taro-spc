# 自定义快捷键

| 设置键      | 操作     |      |
| ----------- | -------- | ---- |
| alt+y       | 终端显示 |      |
| alt+n alt+y | 新建终端 |      |
|             |          |      |

# 内置快捷键

| 设置键       | 操作                   |      |
| ------------ | ---------------------- | ---- |
| ctrl+p       | 打开控制台             |      |
| alt+{number} | 指定第{number}个标签页 |      |
|              |                        |      |
